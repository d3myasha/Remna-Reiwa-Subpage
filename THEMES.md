# 🎨 Темы оформления

Все темы поддерживают **тёмный** и **светлый** режим (`bash setup.sh` → выбор режима или `-m dark` / `-m light`).

| № | ID | setup `-t` |
|---|-----|------------|
| Purple (по умолчанию) | 1 |
| Monochrome | 2 |
| Cyberpunk | 3 |
| Emerald | 4 |
| Amber | 5 |
| Ocean | 6 |
| Blush | 7 |
| Open Red | 8 |
| Open Grape | 9 |
| Open Teal | 10 |
| Open Yellow | 11 |

Темы **8–11** используют палитру [Open Color](https://github.com/yeun/open-color).
